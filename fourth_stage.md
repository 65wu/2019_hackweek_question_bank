# 第四章

本章我们会学习python的循环语句
在python中，基本的循环语句有两种,分别是`for`循环和`while`循环
___
### for循环
for循环可以用来把一些集合中的所有元素依次访问一次
for循环的语法格式如下：
```
	for iterating_var in sequence:
  		statements(s)
```
其中 `for`和`in`都是关键字，
`iterating_var`是元素，
`sequence`使我们要遍历的集合，
`statements(s)`是合法的python语句

我们用以下的例子体会一下

样例输入：

		for i in 'NcuHome':
			print i
			
样例输出：
```
N
c
u
H
o
m
e
```
在这个例子中,i是元素，用它来代表字符串'NcuHome'中每个字母
'NcuHome'是我们要遍历的集合
`print(i)`是循环里执行的语句，我们用print把当前的i打印出来
在循环里，i是变动的,它先是从首字母'N'出发，在执行完print(i)后，i又进发字母'c'，在执行完print(i)后，又继续往后，直到终点字母'e'，在把集合中的每一个元素都遍历以后，for循环也便结束了

### range()函数
`range(start, stop[, step])`
range函数共有三个参数
+ start代表计数的起始元素，默认为0
+ stop代表计数的末尾元素，且不包含末尾元素
+ step代表步长，默认为1，表示range()增长的速度

样例输入:

		for i in range(1,10):
			print(i)
			
样例输出：
```
1
2
3
4
5
6
7
8
9
```
在这个样例中,起始元素为1，之后迭代到9，但不包括末尾元素10，并且默认步长为1
于是显示出以上结果
通过range()函数，我们可以很轻松的输出一个有序数组所有的元素，而不用像之前那样列举出所有元素

### while循环
基本语法:
```
while 条件语句:
	执行语句
```
while循环有些类似for循环
它也可以多次执行特定的语句
while循环开始时，会判断条件语句是否成立，如果布尔值为否，循环立马退出不执行；
若布尔值为真，则正常执行下方语句，执行完毕以后回到while循环头部，再次判断条件反复，如此反复，直至
循环退出
___

### 死循环

死循环又称为无限循环，它是由于while条件始终为真导致的
我们把上一题的代码稍作修改
```
count = 0;
while count >=0
	print( '*' )
	count += 1
```
该循环会一直执行下去，直到耗光机器的所有的内存
如果你不小心运行了一个死循环，别担心，你可以使用CTRL + C的方式来强行终止程序

### 循环的嵌套
所谓循环的嵌套，便是指在循环中使用另一个甚至多个其它循环
样例输入：
```
	for i in range(1,10):
		for j in range(1,i+1):
			print("i * j = ",i * j,end="  ")
		print()
```
样例输出：
```

i * j =  5  i * j =  10  i * j =  15  i * j =  20  i * j =  25  
i * j =  6  i * j =  12  i * j =  18  i * j =  24  i * j =  30  i * j =  36  
i * j =  7  i * j =  14  i * j =  21  i * j =  28  i * j =  35  i * j =  42  i * j =  49  
i * j =  8  i * j =  16  i * j =  24  i * j =  32  i * j =  40  i * j =  48  i * j =  56  i * j =  64  
i * j =  9  i * j =  18  i * j =  27  i * j =  36  i * j =  45  i * j =  54  i * j =  63  i * j =  72  i * j =  81  
```
循环开始时,i=1
进入下一条语句 for j in range(1,2)

执行print语句后回到 for j in range(1,2)，j循环直接结束
执行j循环下方语句print()
循环第一遍结束后回到for i in range(1,10)，此时i=2
进入for j in range(1,3)
j值初始化为1
执行print语句后回到 for j in range(1,3),此时j=2
执行print语句后回到 for j in range(1,3)，j循环结束
执行j循环下方语句print()
i循环这一遍结束后回到最上方for i in range(1,10)此时i=3
如此反复，直至i=9时的循环执行完毕

5.请问以下三串代码中，哪一串代码的打印出的几何图形是实心三角形？(A)

A.
```
r = int(input())
for i in range(1,r,2):
    for j in range( int( ( r-i )/2 ) ):
        print(" ",end='')
    for k in range(i+1):
        if k == 0:
            print("/",end='')
        elif k == i:
            print("\\",end='')
        else:
            print("-",end='')
    print("")
```
B.
```
r = int(input())
for i in range(1,r+1):
    for j in range(1,r+1):
        if( (j == 1) | (j == r) )&( (i != 1) & (i != r) ):
            print(" | ",end='')
        elif ( (i == 1) | (i == r) ) & ( (j != 1) & (j != r) ):
            print(" - ",end='')
        else:
            print(" + ",end='')
    print('\n')
```
C.
```
r = int(input())
for i in range(1,r+1):
    for j in range(1,r+1):
        if ( i+j in range( int((r+3)/2) , int((3*r+1)/2) + 1) ) & ( i - j in range( int((1-r)/2) ,int((r-1)/2) + 1) ):
            print("+",end='')
        else:
            print(" ",end='')
    print()
```
错误解释：

A选项为三角形

i为层数深度，j控制空格符宽度，k负责在三角形不同的位置输出特定的字符，r控制三角形大小

先在每一行的左方输出空格，层数越浅空格占位符数量越多，空格输出完毕以后输出三角形部分

三角形部分由三种字符组成，最左端的`/`、中间的`-`,最右端的`\`，并且中间的`-`字符数量会随着的层数增多而增多

B选项为矩形

i为长，j为宽，r负责控制矩形的大小

在矩形上下两排并且不是四角的位置输出`-`字符

在矩形左右两列并且不是四角的位置输出`|`字符

其余位置包括四角输出`+`字符

C选项为菱形

同上i,j,r意义

我们先构造一个长r,宽r的矩形，之后用四条直线从矩形的四条角割掉四个全等的直角三角形

形成菱形，原理属于线性规划知识

之后在四条直线围成的封闭区域内用`+`字符填充，形成菱形
